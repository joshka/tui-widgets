# Tui-cards

<!-- cargo-rdme start -->

A simple library to render playing cards in a terminal using tui-rs.

## Example

```rust
use tui_cards::{Card, Rank, Suit};

let card = Card::new(Rank::Ace, Suit::Spades);
frame.render_widget(&card, frame.area());
```

## Demo

```shell
cargo run --example card
```
